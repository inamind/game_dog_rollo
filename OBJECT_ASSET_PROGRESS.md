# ROLLO 물건 에셋 제작 진행표

## 제작 규칙

- 물건마다 정상·경미 파손·심각 파손·완파의 4단계
- 2단계는 1단계와 축소 화면에서도 즉시 구별되도록 약 25~30%의 큰 균열·결손·부품 이탈을 표현
- 3단계는 원래 구조의 약 50%가 무너지거나 사라진 상태로 표현
- 파일명: `object_<id>_state_<1-4>.png`
- 최종 규격: `512×384px`, 투명 RGBA PNG, 바닥 앵커 통일
- 화풍 기준: `object_sofa_state_1.png`
- 완성 즉시 `OBJECT_SPRITE_PATHS`에 등록하고 프로젝트 검사를 실행
