# ROLLO

강아지 ROLLO를 조작해 마당과 3층 주택의 물건을 부수는 HTML5 Canvas 게임입니다.

## 실행

Node.js가 설치된 환경에서 다음 명령을 실행합니다.

```bash
npm run dev
```

브라우저에서 `http://127.0.0.1:4173/`을 엽니다.

## 조작

- 이동: 방향키 `←`, `→`
- 계단: 방향키 `↑`, `↓`
- 스매쉬: `Q`
- 필살기: `W`
- 태블릿: 화면 좌우 아래의 가상키 사용

## 검사

```bash
npm run check
```

게임 기획과 에셋 규칙은 `ROLLO_게임_기획서.md`, `PROJECT_MANAGEMENT.md`, `OBJECT_ASSET_PROGRESS.md`에서 확인할 수 있습니다.
